# Contextualização do Projeto

O **NotePlus+** é uma aplicação web desenvolvida com o objetivo de organizar, analisar e apresentar de forma estratégica os dados relacionados às notas de corte do Sistema de Seleção Unificada (SISU).

A ideia do projeto surge a partir da dificuldade enfrentada por estudantes ao tentar compreender a dinâmica das notas de corte, que apresentam alta volatilidade durante o período de inscrição. As informações disponíveis costumam estar dispersas, sem ferramentas adequadas de visualização, comparação ou análise histórica.

Diante desse cenário, o NotePlus+ foi concebido como uma solução baseada em **engenharia de software e análise de dados**, permitindo transformar informações públicas em conhecimento estratégico para tomada de decisão.

O sistema busca reduzir a assimetria informacional no acesso ao ensino superior, oferecendo recursos que auxiliam candidatos na escolha mais consciente de cursos e instituições.

---

# Funcionalidades Principais

A aplicação foi projetada para fornecer uma experiência centrada na análise e interpretação de dados educacionais.

Entre as funcionalidades principais, destacam-se:

- Visualização estruturada das notas de corte por curso e instituição  
- Aplicação de filtros dinâmicos para exploração de dados  
- Comparação entre modalidades de concorrência  
- Exibição de tendências históricas das notas de corte  
- Simulações que auxiliam na estimativa de desempenho competitivo  

Esses recursos permitem ao usuário compreender padrões e comportamentos do processo seletivo, contribuindo para decisões mais estratégicas.

# Stack Tecnológica

O desenvolvimento do NotePlus+ envolveu a construção completa de uma aplicação full-stack moderna, com separação clara de responsabilidades entre interface, lógica de negócio e persistência de dados.

## Frontend

- React  
- Mantine  

O frontend é responsável pela renderização da interface, experiência do usuário e interação com os dados disponibilizados pela API.

---

## Backend

- TypeScript  
- TypeORM  

O backend implementa a lógica de processamento das informações, modelagem das entidades e exposição dos endpoints responsáveis pela comunicação com o cliente.

---

## Banco de Dados

- PostgreSQL  

O sistema envolve criação, modelagem e manipulação de banco de dados relacional, garantindo armazenamento consistente das informações e suporte às consultas analíticas realizadas pela aplicação.
